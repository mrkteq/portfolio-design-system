## Development

1. Clone the [repo](https://github.com/mrkteq/design-system): `git@github.com:mrkteq/design-system.git`
2. Run `npm install`
3. Run `npm start` to start the [Eleventy](https://11ty.dev) server
4. Browse to `http://localhost:8080`
